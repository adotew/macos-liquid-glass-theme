# MacOS Liquid Glass

A dark liquid glass theme for [Zed](https://zed.dev) inspired by Apple's design language.

Native macOS window blur meets a solid, readable editor. System blue accents. Built for macOS.

## Install

From the Zed Extension Store:

```
Cmd + Shift + P → zed: extensions → Search "MacOS Liquid Glass"
```

Or manually:

```bash
mkdir -p ~/.config/zed/themes
cp themes/liquid-glass.json ~/.config/zed/themes/
```

Then select **MacOS Liquid Glass** from the theme picker (`Cmd + K, Cmd + T`).
